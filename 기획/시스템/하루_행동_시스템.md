# 하루 행동 시스템 기획

## 1. 목적

이 문서는 날짜 기반 미연시 진행에서 플레이어가 하루 동안 무엇을 할 수 있는지 정의한다.

현재 스토리는 1학기 말 첫 회차에서 D-32부터 시작하고, 루프 이후에는 D-25 월요일부터 내부 기준 D-Day 금요일까지 약 4주를 진행한다. 축제는 D-1 목요일 1일차와 D-Day 금요일 2일차의 이틀 구성이다. 1일차는 부스 운영 중심으로 하연과 `쉬어가 카페`를 함께 운영하며 가까워지고, 2일차는 부스 축소 운영 후 히로인들과 축제를 둘러보는 선택을 거쳐 폐장 후 사고로 이어진다. 단, `D-Day`는 내부 기획/시스템 용어이며 게임 내 대사/UI에서는 `축제 2일차`, `축제 마지막 날`, `금요일`로 표기한다. 달력상 D+3 월요일 방학식이 예정되어 있지만, 실패 루프에서는 그 날짜에 도달하지 못한다. 진엔딩에서는 D-Day 금요일 밤 이후 토요일 날짜를 건너뛰고 일요일 오후 병실에서 깨어난 뒤, D+3 월요일 방학식에 겨우 참석한다. 플레이어는 한정된 시간 안에 다음 세 축을 모두 충족해야 한다.

1. 하연과의 상호 마음 확인
2. 기존 관계 정리
3. 사고 단서 확보

하연과 서로의 마음을 확인하지 못하면, D-Day 금요일 밤 사고 기억에서 하연이 빠지고 재윤은 혼자 야외 소무대에 선 채 사고를 반복한다. 따라서 하연 관련 행동은 단순 호감도 상승이 아니라, 하연이 마음을 말할 수 있을 만큼의 신뢰와 재윤이 그 마음을 받아들일 준비를 쌓는 행동이어야 한다.

따라서 하루 행동 시스템은 단순 호감도 선택이 아니라, 제한된 시간 안에서 감정 / 관계 / 조사 사이의 균형을 선택하게 만드는 구조여야 한다.

D-1 밤 정산과 D-Day 낮 보정은 [D-1/D-Day 조건 수치화](../스토리/12_D-1_D-Day_조건_수치화.md)의 점수 기준을 사용한다. 하루 행동의 누적 결과는 D-Day 폐장 후 `하연 실패 루프`, 기존 관계 새드엔딩, 사고 의미 미연결 루프, 진엔딩 중 하나로 판정된다.

## 1.1 핵심 상태 키

하루 행동 시스템은 아래 키를 기준으로 스토리 상태를 갱신한다. 이름은 [미연시 코어 시스템 요구사항](./미연시_코어_시스템_요구사항.md)의 네이밍 규칙을 따른다.

| 키 | 타입 | 사용처 |
| --- | --- | --- |
| `CurrentDay` | 날짜 ID | 현재 날짜 허브와 이벤트 개방 |
| `CurrentSlot` | 슬롯 ID | 등교/수업/방과후/저녁/주말 슬롯 구분 |
| `HayeonTrust` | int | 하연 신뢰 누적 |
| `HayeonPace` | int | 하연의 조용한 동선 이해 |
| `IsSohaResolved` | bool | 소하 관계 정리 완료 |
| `IsSeorinResolved` | bool | 서린 관계 정리 완료 |
| `IsMiruResolved` | bool | 미루 관계 정리 완료 |
| `HasHayeonClue` | bool | 하연과 사고 기억 연결 단서 |
| `HasSohaClue` | bool | 소하를 통해 얻는 감정 단서 |
| `HasSeorinClue` | bool | 서린을 통해 얻는 소무대/조명 단서 |
| `HasMiruClue` | bool | 미루를 통해 얻는 표지/고정끈 단서 |
| `Avoid` | int | 선택 회피 누적 |
| `CanEnterDDayTrueEnding` | bool | D-Day 진엔딩 선택지 개방 가능 |
| `IsDDayTrueEnding` | bool | D-Day 진엔딩 분기 통과 |
| `HasSeenTrueEndingWake` | bool | D+2 병실 기상 컷신 완료 |
| `HasReachedTrueEndingMonday` | bool | D+3 월요일 도달 |
| `CanEnterTrueEndingCredit` | bool | 진엔딩 크레딧 진입 가능 |

## 2. 기본 원칙

- 평일과 주말의 플레이 감각을 구분한다.
- 평일은 학교 루틴 안에서 작은 선택을 쌓는 구조다.
- 주말은 큰 이벤트, 데이트, 조사, 클라이맥스가 발생하는 특별한 날이다.
- 모든 행동은 최소 하나 이상의 상태에 영향을 준다.
- 하연만 따라가도, 기존 히로인만 따라가도, 조사만 해도 진엔딩은 열리지 않는다.

## 3. 평일 구조

평일은 학교에 가는 날이다.

기본 슬롯:

```text
등교
수업
하교 / 방과후
저녁
```

### 3.1 등교 슬롯

등교 슬롯은 하루의 분위기를 정하는 짧은 구간이다.

주요 역할:

- 날짜 시작
- 전날 선택 결과 반영
- 짧은 히로인 접점
- 기억과 감정의 어긋남 암시

가능 이벤트:

| 이벤트 | 설명 | 주요 효과 |
| --- | --- | --- |
| 소하와 등교 | 소꿉친구 관계를 자연스럽게 보여줌 | 소하 관계 / 과거 축 |
| 하연을 멀리서 봄 | 하연에게 끌리는 감정 강화 | 하연 관심 |
| 서린의 공지 | 축제 일정, 방송반/학생회 일정 전달 | 일정 정보 |
| 미루의 인사 | 미루가 주인공을 따르는 모습 | 미루 관계 |
| 이상 현상 | 날짜 표시 오류, 금속음, 기시감 | 기억 균열 암시 |

등교 슬롯은 대부분 자동 이벤트지만, 특정 날짜에는 선택지를 줄 수 있다.

예:

```text
[소하와 같이 간다]
[먼저 학교로 간다]
[하연을 기다려본다]
```

### 3.2 수업 슬롯

수업 슬롯은 큰 선택보다는 상태 변화와 짧은 연출 중심이다.

주요 역할:

- 하연의 존재감 누적
- 다른 히로인의 미세한 반응 표시
- 루프/내면세계 위화감 삽입

가능 이벤트:

| 이벤트 | 설명 | 주요 효과 |
| --- | --- | --- |
| 하연이 답하지 못함 | 전학생의 어색함과 주인공의 관심 | 하연 신뢰 암시 |
| 소하의 쪽지 | 소하가 익숙한 관계를 강조 | 소하 미련 씨앗 |
| 서린의 호출 | 방송반 또는 학생회 업무 연결 | 서린 이벤트 예고 |
| 미루의 실수 소문 | 축제 도움 인원으로서의 서툼 | 미루 이벤트 예고 |
| 졸음 / 짧은 꿈 | 야외 소무대 이미지 침투 | 기억 조각 암시 |

수업 슬롯은 보통 자동 진행으로 두고, 플레이어 피로도를 줄인다.

### 3.3 하교 / 방과후 슬롯

방과후는 평일의 핵심 선택 슬롯이다.

플레이어는 누구와 시간을 보낼지 선택한다.

기본 선택지:

```text
[하연과 축제 준비를 한다]
[소하와 같이 돌아간다]
[서린의 일을 돕는다]
[미루의 멘토링을 봐준다]
[소무대를 조사한다]
[혼자 생각한다]
```

행동별 효과:

| 행동 | 주요 효과 | 리스크 |
| --- | --- | --- |
| 하연과 축제 준비 | 하연 신뢰 상승, 상호 마음 확인 이벤트 진행 | 기존 관계 정리 지연 가능 |
| 소하와 같이 돌아감 | 소하 관계 정리 진행, 과거 단서 | 하연 이벤트 부족 가능 |
| 서린의 일을 도움 | 방송 대본, 안전 안내, 구조물 단서 | 선택 회피/의존 증가 가능 |
| 미루의 멘토링을 봐줌 | 곧 끝나는 멘토링, 미루 죄책감, 고정끈 단서 | 의존 심화 가능 |
| 소무대 조사 | 사고 단서 직접 획득 | 감정 조건 부족 가능 |
| 혼자 생각 | 기억 조각 정리, 기억 혼선 완화 | 관계 진행 부족 |

### 3.4 저녁 슬롯

저녁 슬롯은 하루 정산과 기억 균열 연출 구간이다.

주요 역할:

- 오늘 선택 결과 정리
- 기억 조각 획득 연출
- 기억 혼선 변화
- 관계 균열 단계 업데이트
- 악몽 또는 짧은 현실 신호 삽입

가능 연출:

- 금속이 삐걱이는 소리
- 병원 기계음처럼 들리는 알람
- 오늘 찍은 사진에서 하연만 흐릿함
- 소하와의 기억이 하연과의 기억을 덮어씀
- 서린의 일정표가 선택지처럼 보임
- 미루의 메시지가 읽지 않았는데 읽음 처리됨

저녁 슬롯은 보통 자동 정산이지만, 특정 조건에서는 선택지를 제공한다.

예:

```text
[오늘 일을 일기에 적는다]
[기억 조각을 확인한다]
[그냥 잠든다]
```

## 4. 주말 구조

주말은 학교와 직접 관련 없는 특별한 날이다.

평일의 축제 준비 / 학생회 / 등하교 루틴에서 벗어나, 인물의 사적인 면과 관계의 깊이를 보여준다.

기본 슬롯:

```text
오전
오후
밤
```

주말은 다음 용도로 사용한다.

- 학교 밖 데이트 / 외출
- 우연한 만남
- 만나지 않고 혼자 보내는 날
- 약속 이벤트
- 봉사활동 / 가족 심부름 / 동네 행사
- 루프 후 기억 조각 반응

중요한 제한:

> 주말에는 원칙적으로 학교 무대 준비나 소무대 작업을 하지 않는다.

학교 관련 사고 단서는 평일 방과후, 학생회 업무, 축제 준비 시간, 또는 별도 조사 행동에서 다룬다.

## 5. 주말 행동 종류

### 5.0 주말 메신저 선택 구조

특정 주말에는 아침에 여러 연락이 동시에 와 있는 구조를 사용할 수 있다.

예:

```text
2학년 5반 단톡방
소하 개인 메시지
방송반 단톡방
미루 개인 메시지
```

처리 원칙:

- 플레이어는 각 연락을 모두 읽어볼 수 있다.
- 실제로 갈 곳은 하나만 선택한다.
- 하나를 선택하면 나머지는 자동으로 거절하거나 다음으로 미룬다.
- 자동 거절은 관계 단절이 아니라 하루 시간 제약을 표현하는 장치다.
- 선택한 연락만 해당 날짜의 관계/후보/단서 씨앗을 강화한다.

### 5.1 하연과 외출

하연과 학교 밖에서 긴 시간을 보내는 이벤트.

가능 내용:

- 서점, 조용한 카페, 공원, 강변 산책
- 하연이 시끄러운 장소보다 조용한 장소를 편하게 느끼는 이유를 보여줌
- 말이 적어도 함께 있는 시간이 불편하지 않다는 감정선 강화

효과:

- 하연 신뢰 큰 폭 상승
- 하연과의 상호 마음 확인 이벤트 진행
- 하연의 현실적인 매력 강화

리스크:

- 다른 관계의 미련과 균열이 배경에서 진행됨

### 5.2 소하와 추억의 장소 방문

소하 루트의 큰 이벤트.

가능 내용:

- 어릴 때 자주 갔던 문방구, 놀이터, 동네 분식집
- 오래된 사진 / 팔찌 / 약속
- 주인공은 익숙함을 느끼지만, 그 익숙함이 하연에게 향한 마음을 덮으려 한다는 위화감도 느낀다.

효과:

- 소하 관계 정리 진행
- 감정 단서 획득

리스크:

- 하연과의 마음 확인 진행 지연

### 5.3 서린과 학교 밖 업무 / 상담

서린 루트의 주말 이벤트는 학교 무대 준비가 아니라, 학교 밖에서 발생하는 방송반 / 학생회 관련 외부 업무나 상담으로 처리한다.

가능 내용:

- 축제 협찬 물품 수령
- 문구점 / 인쇄소 / 대여 업체 방문
- 학생회 자료 정리 카페 회의
- 방송 대본 출력물 확인
- 안전 안내 방송 문구 검토
- 서린이 주인공에게 선택과 책임에 대한 조언을 함

효과:

- 서린과의 현실 관계 강화
- 서린이 왜 주인공에게 `정답을 알려주는 사람`처럼 각인되었는지 보여줌
- 안전 점검 단서의 간접 예고 가능
- 방송/안내가 선택지 통제 불안으로 변질될 씨앗 제시

리스크:

- 서린의 선택 회피/통제 불안 증가

### 5.4 미루와 봉사활동 / 동네 심부름

미루 루트의 주말 이벤트는 학교 밖에서 주인공이 미루를 챙기는 관계를 보여준다.

미루는 주인공의 `1학기 한정 학습·학교생활 적응 멘토링`을 받던 후배다. 이 멘토링은 수학/영어 보충, 수행평가 정리, 발표 연습, 제출물과 마감일 정리, 봉사활동이나 행사 신청 도움, 모르는 일이 생겼을 때 누구에게 질문하고 보고할지 정하는 연습까지 포함한다. D+3 월요일 방학식이 오면 공식적으로 끝나며, 이후에는 멘토링 관계가 평범한 선후배 관계로 바뀐다. 주말 이벤트는 단순한 데이트보다, 미루가 주인공에게 왜 특별히 의지하게 되었는지와 멘토링이 끝나기 전에 혼자 해보려는 부담을 함께 보여주는 방향이 좋다.

가능 내용:

- 지역 봉사활동
- 동네 행사 보조
- 분실물 찾기
- 무거운 물건 옮기기
- 미루가 실수하고 주인공이 도와줌
- 수행평가 자료 찾기
- 발표 연습을 봐주기
- 봉사활동 신청서를 같이 작성하기
- 축제 도움 인원 체크표 만들기
- 실수했을 때 보고할 사람 정하기
- 물어봐도 되는 일과 혼자 해볼 일을 나누기

효과:

- 미루와의 의존 관계 강화
- D+3 월요일 방학식과 함께 1학기 한정 멘토링이 끝난다는 불안
- 미루의 서툼과 죄책감의 씨앗을 자연스럽게 보여줌
- 미루가 축제 도움 인원에 참여하게 된 계기 제시

리스크:

- 미루의 의존 심화

### 5.5 우연한 만남

플레이어가 특정 약속을 잡지 않았거나 혼자 외출했을 때 발생할 수 있다.

가능 예시:

- 서점에서 하연과 우연히 만남
- 동네 분식집에서 소하와 마주침
- 버스 정류장에서 서린과 만남
- 봉사활동 현장에서 미루와 마주침

효과:

- 선택하지 않은 캐릭터도 세계 안에 살아 있다는 느낌 제공
- 낮은 확률 또는 조건부 보너스 이벤트로 활용

### 5.6 혼자 보내기 / 기억 정리

플레이어가 아무도 만나지 않고 하루를 보내는 행동.

가능 내용:

- 집에서 쉬기
- 일기 쓰기
- 기억 조각 정리
- 꿈속 장면 복기
- 산책 중 금속음이나 병원 신호를 떠올림

효과:

- 기억 혼선 완화
- 기억 조각 힌트 제공
- 루프 후 빠른 진행 도움

리스크:

- 관계 진행 기회 소모

## 6. 수업 이벤트 처리

수업은 1교시부터 여러 교시까지 모두 플레이하지 않는다.

평소 수업은 축약하고, 특정 이벤트가 있을 때만 `수업 이벤트`로 보여준다.

### 기본 처리

일반적인 날의 수업은 한두 문장으로 넘긴다.

예:

> 오전 수업은 별일 없이 지나갔다.  
> 창밖 운동장에는 축제 천막이 하나둘 세워지고 있었다.

### 수업 이벤트가 발생하는 경우

| 유형 | 설명 | 예시 |
| --- | --- | --- |
| 하연 이벤트 | 하연의 전학생 적응, 짧은 반응 | 하연이 대답을 망설이자 주인공이 대신 분위기를 풀어줌 |
| 소하 이벤트 | 소하의 익숙한 관계 강조 | 소하가 주인공의 과거 답안을 기억하고 있음 |
| 서린 이벤트 | 학생회/방송/공지 개입 | 수업 중 방송으로 서린의 목소리가 나오고, 특정 문장이 반복됨 |
| 미루 이벤트 | 후배의 실수 소문 | 복도에서 1학년 도움 인원이 실수했다는 이야기가 들림 |
| 기억 균열 이벤트 | 내면세계 위화감 | 칠판 날짜가 바뀌거나, 종소리가 병원 기계음처럼 들림 |
| 루프 이벤트 | 반복 감각 | 선생님이 지난 회차와 같은 말을 하지만 일부 단어가 달라짐 |

### 수업 중 깜짝 이벤트 후보

수업 이벤트는 짧고 인상적으로 처리한다.

후보:

1. 칠판 날짜 오류
   - 수업 시작 때 칠판에는 `D-14`라고 적혀 있었는데, 눈을 깜빡이면 `D-7`로 보임.
2. 종소리 변조
   - 수업 종료 종이 잠깐 심전도 소리처럼 들림.
3. 하연 이름 공백
   - 출석부에서 하연 이름만 흐릿하게 보임.
4. 반복 문장
   - 선생님이 `중요한 건 선택을 미루지 않는 것` 같은 문장을 이상할 정도로 강조함.
5. 책상 수 불일치
   - 교실에 책상이 하나 많거나 적어 보임.
6. 소무대 낙서
   - 교과서 여백에 자신이 쓴 적 없는 `소무대에 가지 마`라는 글씨가 있음.
7. 순간 암전
   - 형광등이 깜빡이고, 교실 천장이 야외 조명 프레임처럼 보임.
8. 방송 잡음
   - 학교 방송에서 축제 안내가 나오다가 금속이 긁히는 소리로 변함.

### 수업 이벤트 설계 원칙

- 한 수업 이벤트는 짧게 끝낸다.
- 하루에 여러 교시를 모두 플레이하지 않는다.
- 중요한 날에만 특정 교시를 명명한다. 예: `3교시 국어`, `5교시 자습`.
- 수업 이벤트는 상태값을 크게 바꾸기보다 위화감, 힌트, 다음 행동 유도에 사용한다.

## 7. 주차별 행동 변화

### Week 1: D-25 ~ D-19

평일:

- 선택지는 비교적 평범함
- 하연과의 첫 대화와 기존 히로인의 미묘한 반응 중심
- D-23 수요일에 담임이 축제 이틀 일정과 2학년 5반의 반 단위 참여를 공식 공지하고 최종 목표가 D-Day로 고정됨
- D-23 수요일에 축제 2일차 뒤 D+3 월요일 방학식이 있다는 일정도 함께 공지되어, 첫 회차에는 평범한 학기말 일정으로 보이고 루프 이후에는 도달하지 못하는 다음 월요일처럼 느껴짐

주말:

- 하연과 첫 긴 대화 가능
- 소하/서린/미루 중 하나와 큰 이벤트 가능

### Week 2: D-18 ~ D-12

평일:

- 축제 준비가 본격화됨
- D-18 월요일에 2학년 5반 반행사를 `쉬어가 카페`로 확정하고, 이후 평일 방과후 선택에서 축제 준비 접점이 본격화됨
- D-18 이후부터는 선택 결과가 세계에 반영되는 감각과 아주 약한 어긋남을 서서히 늘린다.
- 소무대 조사, 서린 업무, 미루 정리 작업 선택지가 중요해짐

주말:

- 사고 단서 집중 이벤트 가능
- 기존 관계의 미련과 균열이 뚜렷해짐

### Week 3: D-11 ~ D-5

평일:

- 관계 균열이 뚜렷해짐
- 사고 단서가 구체화됨
- 소무대 조사, 서린 점검표, 미루 고정끈 관련 이벤트 중요도 상승

주말:

- 학교 밖 큰 이벤트
- 히로인 클라이맥스 진입 조건 확보

### Final Week: D-4 ~ D-1

평일 / 축제 1일차:

- 기억 회복과 불안 연출 강화
- 선택하지 않은 관계의 미련과 균열도 배경에서 진행
- D-1에는 하연과의 부스 운영, 기존 관계 정리, 사고 단서 최종 확인을 통해 D-Day 조건을 맞추기 위한 마지막 조정
- D-1 목요일은 축제 1일차로, 2학년 5반 `쉬어가 카페` 등 부스가 정상 운영됨
- D-1에는 재윤과 하연이 같은 부스 운영 시간대에 묶여 함께 일하며 가까워지고, 축제 2일차에는 약속 예약이 아니라 재윤이 하연의 동선과 속도를 읽는 구조로 이어짐
- D-Day 진입 전 최종 정산과 하연의 말하지 못한 예감 이해 여부 확인

### D+3 월요일: 방학식 예정

- D-1 목요일 축제 1일차와 D-Day 금요일 축제 2일차 뒤 주말을 지나 예정된 1학기 방학식
- 실패 루프에서는 일반 날짜 진행으로 도달하지 못하고 D-25 월요일로 돌아감
- 진엔딩에서는 `DDay_05True` 이후 토요일 허브 없이 `TE_00_TimeSkip`로 암전 통과하고, 일요일 오후 병실 Scene인 `TE_01_Wake`로 점프함
- D+2 병실 기상 후 `HasSeenTrueEndingWake`를 켜고, D+3 병원 출발 시 `HasReachedTrueEndingMonday`를 켠 뒤 방학식 에필로그 Scene으로 진입함
- 방학식 이후 방학 약속 컷을 지나 `CanEnterTrueEndingCredit`가 켜지면 진엔딩 크레딧으로 연결함
- 재윤은 큰 외상은 없지만 알 수 없는 이유로 깨어나지 못했던 상태이며, 월요일 방학식에는 오래 머물 수 없는 조건으로 짧게 참석함
- 미루의 1학기 한정 멘토링 종료일과 연결되어, 미루 루트에서는 `멘토링이 끝나면 선배와 이어지는 이유도 사라진다`는 불안을 강화함
- 진엔딩 이후 에필로그나 회복 장면에서는 처음으로 도달할 수 있는 날짜 후보로 사용할 수 있음

## 8. 루프 후 행동 변화

루프 후에는 같은 일상을 그대로 반복하지 않는다.

변화:

- 첫 회차 전용 1주 프롤로그 스킵
- 이미 본 이벤트는 압축판 제공
- 기억 조각 보유 시 조사 선택지 조기 개방
- 같은 새드엔딩 반복 시 해당 루트 압축 / 특수 대사 적용

예:

첫 회차:

```text
[소무대를 둘러본다]
```

루프 후, `금속음` 조각 보유:

```text
[소무대의 임시 조명 프레임을 확인한다]
```

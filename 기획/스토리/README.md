## 현재 확정된 핵심

- 장르: 미연시처럼 시작하는 루프 로맨스 미스터리 비주얼노벨
- 실제 정체: 축제 사고 후 혼수상태에 빠진 주인공의 내면세계
- 첫 회차: 1학기 말 D-32 월요일 시작, 기존 히로인 관계를 보여주는 1주 프롤로그 + 하연 등장 후 약 4주
- 은하연 전학: D-25 월요일
- 축제 이틀 일정 공지: D-23 수요일, 2학년 5반의 반 단위 참여 확정
- 2학년 5반 반행사: D-18 월요일 `쉬어가 카페`로 확정
- 축제 1일차: D-1 목요일. 부스 운영 중심, 하연과 `쉬어가 카페` 운영으로 관계 강화
- D-2 수요일: 하연이 조용한 곳에서 마음을 정리하는 사람임을 재윤이 이해하기 시작하는 `말하지 못한 예감`. 하연은 직접 약속을 잡지 않고, 재윤이 D-3 관계 정리 이후 하연의 침묵과 동선을 스스로 읽어야 한다.
- 축제 2일차 / 사고: D-Day 금요일. 부스 축소 운영 후 히로인들과 축제를 둘러보고, 하연 공략이 충분하면 폐장 후 재윤이 하연의 동선을 알아차려 직접 소무대로 찾아가 마음과 선택을 말한다. 부족하면 하연이 사고 기억에서 빠지고, 재윤은 혼자 소무대에 선 이유를 잊은 채 루프한다.
- 방학식 예정: D+3 월요일. 실패 루프에서는 도달하지 못하는 다음 월요일
- 현재 날짜별 상세 확정 범위: D-32 월요일부터 D-Day 금요일, 진엔딩 후 D+3 월요일 방학식까지
- 루프 이후: 첫 회차 전용 1주를 건너뛰고 D-25 하연 등장 직전부터 재시작
- 진엔딩 조건: 하연과의 상호 마음 확인, 기존 관계 정리, 사고 단서 확보. 핵심은 하연을 충분히 이해해 폐장 후 동선을 알아차리고, 미루던 선택과 기억을 받아들인 재윤이 먼저 하연을 찾아가 말하는 것
- 진엔딩: 금요일 밤 사고 후 D+1 토요일은 날짜 카드로 보여주지 않고, D+2 일요일 오후 병실에서 깨어난 뒤 D+3 월요일 방학식에 겨우 참석해 현실의 은하연과 다시 만남. 방학식 직후에는 병원 재검사, 도서관, 카페, 산책 같은 작은 약속으로 방학 동안 두 사람이 계속 만날 것을 암시하고, 음악과 짧은 애니메이션 컷이 들어간 엔딩 크레딧으로 마무리함
- 하연 실패 루프: 하연과 서로의 마음을 확인하지 못하면, 재윤은 하연이 있던 사고 기억을 잃고 혼자 야외 소무대에서 사고를 반복함. 방학식과 완성된 크레딧에는 도달하지 않음
- 이야기의 핵심 질문: 다치지 않은 과거로 돌아갈 것인가, 다친 기억까지 안고 서로 선택한 마음을 인정할 것인가
- 루프 이유: 재윤이 하연을 좋아했다는 마음과 그 선택 끝에 다쳤다는 기억을 동시에 받아들이지 못하기 때문에, 금요일 다음 월요일로 넘어가지 못함
- 비진엔딩 처리: 진엔딩만 방학이 실제로 시작된다. 소하/서린/미루 새드엔딩은 각각 왜곡·정지·닫힘의 가짜 크레딧으로 처리하고, 히든 붕괴는 이름과 음악이 깨지는 크레딧으로 처리한다.

## 현재 스토리 완성 기준

- 하연 루트는 서로 마음을 확인할 수 있는 신뢰를 형성하는 과정이다.
- D-2는 하연이 먼저 마음을 말하거나 만남을 예약하는 날이 아니라, 재윤이 하연의 속도와 조용한 자리 선호를 이해하는 `말하지 못한 예감`이다.
- 축제 마지막 날 폐장 후 성공 분기에서는 하연이 먼저 약속을 잡지 않는다. 재윤이 하연 공략으로 쌓은 이해를 바탕으로 하연이 있을 곳을 알아차려 직접 찾아가 자기 마음과 선택을 말하고, 하연은 자신도 기다리고 있었다는 마음을 확인한다.
- 하연 실패 루프에서는 하연이 사고 기억에서 빠지고, 재윤은 누구에게 무엇을 말하려 했는지 모른 채 혼자 야외 소무대 사고를 반복한다.
- 소하/서린/미루 이벤트는 실제 연애 결론이 아니라, 재윤이 하연에게 가기 전 숨을 수 있는 과거/정답/의존의 길을 정리하는 과정이다. D-3에는 세 관계의 클라이맥스가 발생하며, 각 성공 분기는 진엔딩에 필요한 감정 단서와 사고 단서를 제공한다.
- 사고 단서는 사고를 없애기 위한 정보가 아니라, 다친 기억까지 안고 현실로 돌아가기 위한 기억 조각이다.
- 게임 내 대사/UI에서는 `D-Day`라고 부르지 않고 `축제 2일차`, `축제 마지막 날`, `금요일`, `폐장 후`처럼 표현한다.

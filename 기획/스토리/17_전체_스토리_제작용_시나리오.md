# 전체 스토리 제작용 시나리오

이 문서는 `00_전체_스토리.md`의 소설 원고를 실제 비주얼노벨 제작 단위로 분해한 구현 기준 문서다.

`00`은 문장, 감정선, 장면 호흡의 정본으로 유지한다. 이 문서는 StoryFlow Scene, VN Shot, Branch, Transition, StoryState 연결을 빠르게 확인하기 위한 제작용 인덱스다.

## 1. 사용 기준

- 소설 원고 원본: `00_전체_스토리.md`
- 읽기용 텍스트본: `00_전체_스토리_소설.txt`
- 날짜/이벤트 마스터: `06_날짜별_진행_달력.md`
- D-Day 조건 수치화: `12_D-1_D-Day_조건_수치화.md`
- D-Day 분기 스크립트: `10_D-Day_축제마지막날_분기.md`, `13_D-Day_스크립트_분할.md`
- 장소/배경 기준: `14_장소와_배경_묘사.md`
- 배경 에셋/프롬프트 기준: `15_배경_에셋_프롬프트.md`
- 진엔딩 후일담 컷신 기준: `16_D+2_D+3_진엔딩_후일담_컷신.md`
- 시스템 연결 기준: `../시스템/미연시_코어_시스템_요구사항.md`

## 2. StoryFlow 연결 기준

| 제작 항목 | StoryFlow / VisualNovelPlugin 기준 |
|---|---|
| StorySceneAsset | 큰 흐름 단위 에셋. 예: `SF_Prologue`, `SF_DDay`, `SF_TrueEnding` |
| SceneID | `FStorySceneID`에 들어갈 짧은 PascalCase ID. 예: `DDay_05True`, `TE_01_Wake` |
| ShotID | 실제 대사, 배경 전환, BGM, 컷인 단위. 이 문서에서는 세부 Shot을 번호로 확정하지 않고 주요 연출 단위만 적는다. |
| Branch | 선택지 또는 조건 평가 지점. D-Day는 `DDay_04Gate`와 `EvaluateDDayResult()` 기준을 따른다. |
| Transition | `NextSceneID` 또는 `@goto`에 해당한다. 이 문서의 `다음` 값이 기본 전환이다. |
| StoryState | 저장/조건 키. 시스템 문서의 짧은 PascalCase 규칙을 따른다. 문장형 플래그를 새로 만들지 않는다. |

### SceneID 네이밍 규칙

- `ST_장번호_순번_문장형이름`은 사용하지 않는다.
- 공통 형식은 `그룹_순번_핵심어`다.
- D-Day와 진엔딩 후일담은 이미 정리된 ID를 재사용한다.
- 예시: `Pro_01After`, `Intro_01Wake`, `Hayeon_03Window`, `DDay_05True`, `TE_01_Wake`.

### StorySceneAsset 묶음

| Asset | 포함 SceneID | 역할 |
|---|---|---|
| `SF_Prologue` | `Pro_01After`, `Pro_02Accident` | 현실 사고와 혼수상태 진입 |
| `SF_Intro` | `Intro_01Wake`, `Intro_02Relations`, `Intro_03Crack` | 내면세계의 평범한 시작과 위화감 제시 |
| `SF_Hayeon` | `Hayeon_01Transfer`, `Hayeon_02Lunch`, `Hayeon_03Window`, `Hayeon_04Stage`, `Hayeon_05Premonition` | 하연 공략선과 상호 마음 조건 |
| `SF_Cafe` | `Cafe_01Schedule`, `Cafe_02Name`, `Cafe_03Build`, `Cafe_04DayOne` | 쉬어가 카페 준비와 축제 1일차 |
| `SF_Relations` | `Soha_01OldRoad`, `Soha_02Done`, `Seorin_01Broadcast`, `Seorin_02Choice`, `Miru_01Question`, `Miru_02Room`, `Rel_01Cleanup` | 기존 관계 정리와 단서 확보 |
| `SF_DDay` | `DDay_00Entry`, `DDay_01Choice`, `DDay_03Assess`, `DDay_04Gate` | 축제 마지막 날 선택과 결과 평가 |
| `SF_DDayTrue` | `DDay_05True` | 폐장 후 진엔딩 사고 기억 복구 |
| `SF_TrueEnding` | `TE_00_Sat`~`TE_10_Credit` | 현실 복귀, 방학식, 크레딧 진입 |

| `TE_Unlocked` | 진엔딩 후일담 컷신 진입 가능 | `DDay_05True` |
| `TE_Wake` | 일요일 오후 병실 기상 완료 | `TE_01_Wake` |
| `TE_Monday` | 월요일 방학식 에필로그 도달 | `TE_04_Depart`~`TE_05_Class` |
| `TE_Credit` | 진엔딩 크레딧 진입 가능 | `TE_09_Promise`~`TE_10_Credit` |
| `CeremonySeen` | 방학식 참석 완료 | `TE_07_Hayeon`~`TE_08_Ceremony` |

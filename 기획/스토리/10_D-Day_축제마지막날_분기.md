# D-Day 축제 마지막 날 분기

이 문서는 내부 기준 D-Day 금요일, 게임 내 표기 `축제 2일차 / 축제 마지막 날 / 폐장 후`의 낮 선택과 폐장 후 성공·실패 분기를 대본 수준으로 정리한다. 점수·플래그 판정 기준은 [D-1/D-Day 조건 수치화](./12_D-1_D-Day_조건_수치화.md)를 따르고, 실제 제작 단위 분할은 [D-Day 스크립트 분할](./13_D-Day_스크립트_분할.md)을 따른다.

장소와 배경의 시각 기준은 [장소와 배경 묘사](./14_장소와_배경_묘사.md)를 따른다. 이 문서의 장소 지시는 감정 대본용 요약이며, 배경 원화/AI 이미지 생성 기준은 14번 문서를 우선한다.

D-Day의 핵심은 사고를 피하는 것이 아니다. 재윤이 하연을 충분히 이해했는지, 기존 관계 뒤에 숨지 않았는지, 사고 단서를 `하연에게 가려던 선택의 결과`로 받아들일 수 있는지 확인하는 최종 분기다.

## 장면 목적

| 목적 | 설명 |
| --- | --- |
| 낮 선택 정산 | 하연 신뢰, 기존 관계 정리, 사고 단서를 마지막으로 보정하거나 확정한다. |
| 하연 동선 인식 | 하연이 부르지 않아도 재윤이 폐장 후 하연이 있을 곳을 알아차리게 한다. |
| 선고백 | 재윤이 먼저 자기 마음과 선택을 말한다. |
| 사고 의미 회복 | 사고는 사라지지 않지만, 혼자 다친 사고가 아니라 하연에게 가려던 선택의 결과로 기억된다. |
| 실패 루프 | 조건 부족 시 하연이 사고 기억에서 빠지고, 재윤은 혼자 소무대에 선 이유를 잊은 채 루프한다. |

## 진입 조건

기본 진입:

- D-1 축제 1일차 `쉬어가 카페` 운영 완료.
- D-2 `말하지 못한 예감` 장면을 봄.
- D-3 관계 정리 결과가 정산됨.

주요 조건:

| 조건 | 성공 쪽 의미 | 부족 시 영향 |
| --- | --- | --- |
| `HaYeonTrust` | 하연이 자신의 침묵을 재윤에게 맡길 수 있음 | 하연의 말이 단순 혼잣말처럼 사라짐 |
| `HaYeonPaceRead` | 폐장 후 하연이 조용한 곳으로 향할 것을 재윤이 알아차림 | 하연이 보이지 않아도 어디로 갔는지 모름 |
| `SoHaResolved` | 과거 도피 차단 | 하연에게 가는 마음이 새 관계 도피처럼 흔들림 |
| `SeoRinResolved` | 정답 의존 차단 | 위험하니 가지 않는 것이 정답처럼 보임 |
| `MiRuResolved` | 필요한 사람 도피 차단 | 누군가를 두고 가는 죄책감으로 선택을 미룸 |
| 사고 단서 | 소무대 위험과 병실 기억 연결 | 장소와 물건만 남고, 누구에게 가려 했는지가 비어 있음 |

## 전체 흐름 요약

1. 오전~이른 오후: `쉬어가 카페` 축소 운영.
2. 낮 선택: 하연/소하/서린/미루/혼자 중 하나를 선택해 마지막 보정.
3. 폐장 직전: 조건 정산. 하연이 조용히 사라짐.
4. 성공: 재윤이 하연의 동선을 알아차려 소무대로 직접 찾아감.
5. 실패: 장소와 단서만 따라가지만 하연이 사고 기억에서 빠짐.
6. 성공 분기: 선고백, 하연 마음 확인, 사고 의미 회복, 일요일 병실로 점프.
7. 실패 분기: 혼자 소무대, 사고 반복, D-25 월요일로 루프.

## 1. 오전 공통: 쉬어가 카페의 마지막 운영

### 장소와 분위기

- 2학년 5반 `쉬어가 카페`
- 전날보다 손님은 적고, 부스는 오전~이른 오후까지만 축소 운영
- 컵, 메뉴판, 문구 카드, 손님용 의자가 조금씩 정리되는 중
- 교실 밖은 축제 마지막 날의 들뜬 소음이 있음
- 하연은 카페가 끝나가는 모습을 조용히 바라본다
- 책상을 붙여 만든 임시 카운터, 종이컵, 손글씨 메뉴판, 반쯤 떼어진 종이 장식이 보여야 한다.
- 카페는 상업적으로 예쁜 공간이 아니라 학생들이 만든 어설픈 임시 부스다.
- 창문 쪽에는 운동장 빛이 들어오지만, 교실 안은 이미 정리의 기운이 섞인다.

입구 문구:

```text
잠깐 앉아도 괜찮습니다.
```

재윤은 그 문구를 본다.

내레이션:

> 오늘은 그 문장이, 재윤에게 하는 말처럼 보였다.  
> 쉬어가도 되지만, 머물러 숨으면 안 되는 날이었다.

### 공통 대화

```text
반 친구: 오후엔 부스 닫고 돌아다녀도 된대!
반 친구: 컵 남은 거 이쪽에 모아줘.
재윤: 알겠어.
하연: 이 문구, 떼지 말까?
재윤: 왜?
하연: 아직 들어올 사람 있을 수도 있으니까.
재윤: 축제 끝나기 전까지?
하연: 응. 끝나기 전까지.
```

재윤은 `끝나기 전까지`라는 말을 오래 듣는다.

```text
재윤: 끝나면?
하연: 끝나면... 정리해야지.
```

하연은 그렇게 말하고 메뉴판 쪽으로 시선을 돌린다. 이 시점에서 고백이나 약속은 없다.

## 2. 낮 선택 진입

카페 축소 운영이 끝나면 선택지가 열린다.

```text
[하연과 축제를 둘러본다]
[소하와 마지막으로 오래된 길을 걷는다]
[서린의 방송 마무리를 돕는다]
[미루의 도움 인원 마무리를 확인한다]
[혼자 야외 소무대 쪽을 본다]
```

선택지는 단순 데이트 선택이 아니라, 밤 분기 직전의 마지막 보정이다.

| 선택 | 보정/확정 | 위험 |
| --- | --- | --- |
| 하연 | 상호 신뢰, 하연 동선 이해, 말하지 못한 예감 보강 | 기존 관계 정리 부족은 해결하지 못함 |
| 소하 | 과거 미련 최종 확인 | 하연 신뢰 부족은 해결하지 못함 |
| 서린 | 선택권/위험 단서 최종 확인 | 마음 확인 부족은 해결하지 못함 |
| 미루 | 의존/도움 인원 단서 최종 확인 | 하연 동선 이해 부족은 해결하지 못함 |
| 혼자 | 사고 단서 직접 보강 | 단서 공유가 없으면 하연이 기억에서 빠질 위험 |

## 3. 낮 선택: 하연과 축제를 둘러본다

### 성공/보정 목적

- `HaYeonTrust` 최종 보정
- `HaYeonPaceRead` 최종 보정
- D-2의 예감을 D-Day 낮에 다시 확인

### 대본

축제 마지막 날의 운동장. 음악은 밝고 사람들은 들떠 있다. 하연은 재윤 옆을 걷지만, 축제 중심으로 깊게 들어가지는 않는다.

배경은 운동장 중심의 밝은 천막과 가장자리의 조용한 그늘을 함께 보여준다. 하연은 축제 중심부보다 사람 흐름이 느슨한 가장자리에 시선이 머무르고, 멀리 소무대 방향이 작은 프레임처럼 보이면 좋다.

```text
재윤: 사람 많네.
하연: 어제보다 더 많은 것 같아.
재윤: 괜찮아?
하연: 괜찮아. 오래 있으면 조금 시끄럽긴 한데.
```

재윤은 바로 `그럼 나갈까?`라고 묻지 않는다. 대신 하연의 걸음이 느려지는 방향을 본다.

```text
재윤: 저쪽으로 돌까?
하연: 어디?
재윤: 사람 덜 몰리는 쪽.
```

하연은 잠깐 재윤을 본다.

```text
하연: 알아봤네.
재윤: 조금은.
하연: 조금이면 됐어.
```

두 사람은 운동장 가장자리로 걷는다. 멀리 학교 뒤편 소무대가 보인다.

```text
재윤: 이상하게 계속 저쪽이 걸려.
하연: 나도 가끔 봐.
재윤: 왜?
하연: 시끄러운 데서 잠깐 빠져나가고 싶을 때, 저쪽이 제일 조용해 보여서.
```

D-2 성공 시 추가 대사:

```text
재윤: 혼자 있고 싶은 거랑, 아무도 오지 않았으면 하는 건 다르다고 했잖아.
하연: 기억했네.
재윤: 응.
하연: 그럼 됐어.
```

이 장면에서는 고백하지 않는다. 낮에는 아직 축제 속에 있고, 말해야 할 시간은 폐장 후에 온다.

### 결과

성공:

- `HaYeonTrust +1`
- `HaYeonPaceRead +1`
- 폐장 후 하연 동선 인식 강화

실패/약한 진행:

- 재윤이 하연의 시선을 보지 못하고 축제 놀이에만 집중하면 동선 이해 보정 없음
- 하연이 `괜찮아`만 반복하고, 소무대 방향이 의미 없이 지나감

## 4. 낮 선택: 소하와 마지막으로 오래된 길을 걷는다

### 성공/보정 목적

- `SoHaResolved` 최종 확인 또는 보정
- 과거 도피 잔류 여부 확정

### 대본

소하는 2학년 2반 부스 정리를 마치고 재윤을 부른다.

```text
소하: 야, 유재윤.
재윤: 정리 끝났어?
소하: 대충. 넌?
재윤: 우리도 거의.
소하: 그럼 잠깐 걸어.
```

두 사람은 운동장 옆 오래된 길로 걷는다. 축제 소리는 뒤에서 들리고, 이 길은 이상할 만큼 프롤로그의 등굣길과 닮아 있다.

D-3 소하 정리 성공 시:

```text
소하: 예전 같으면, 너 이런 날엔 나랑 먼저 돌았어.
재윤: 알아.
소하: 그럼 오늘은?
재윤: 마지막에 가야 할 데가 있어.
소하: 하연이한테?
재윤: 응.
소하: 이제야 제대로 말하네.
```

소하는 웃으려다 한숨을 쉰다.

```text
소하: 짜증나는데, 그래도 그게 낫다.
재윤: 뭐가?
소하: 네가 모른 척하는 것보다.
```

D-3 소하 미정리 시:

```text
소하: 아직도 말 안 할 거야?
재윤: 뭘.
소하: 그 애한테 가고 싶은 거.
재윤: 지금 그런 얘기 할 때는...
소하: 너는 항상 그런 식이야. 말할 때가 아니래.
```

보정 선택지:

```text
[하연에게 갈 거라고 말한다]
[예전처럼 웃어넘긴다]
```

성공 시:

```text
재윤: 하연이한테 갈 거야.
소하: ...그래.
재윤: 그래도 너랑 있던 시간이 가짜였다는 뜻은 아니야.
소하: 그건 내가 정해.
재윤: 응.
소하: 근데 지금은 가. 또 늦지 말고.
```

실패 시:

```text
재윤: 그냥 축제잖아.
소하: 그래. 그냥 축제지.
```

소하의 웃음이 프롤로그의 아침처럼 고정된다.

### 결과

성공:

- `SoHaResolved = true` 또는 유지
- 과거 도피 선택지 약화

실패:

- D-Day 밤에 `[아무 일도 없었던 것으로 한다]` 선택지가 강해짐
- 소하 새드엔딩 진입 위험 상승

## 5. 낮 선택: 서린의 방송 마무리를 돕는다

### 성공/보정 목적

- `SeoRinResolved` 최종 확인 또는 보정
- 폐장 방송/소무대 위험 단서 보강
- 정답 의존 잔류 여부 확정

### 대본

방송실. 서린은 폐장 안내 멘트와 장비 회수 큐시트를 확인하고 있다.

방송실은 차가운 통제실이 아니라, 학생회와 방송반 업무가 실제로 쌓인 좁은 공간이다. 믹서, 마이크, 헤드폰, 라벨링된 케이블, 폐장 안내 큐시트, D-9 상담 때 적었던 기준 메모가 함께 보여야 한다.

```text
서린: 왔네.
재윤: 도와드릴 거 있어요?
서린: 네가 도와야 할 일인지, 네가 피하려고 온 일인지 먼저 구분해.
재윤: ...들켰네요.
```

서린은 큐시트를 넘긴다. 큐시트 여백에는 D-9 상담 때 나눴던 네 기준이 작게 적혀 있다.

```text
서린: 폐장 방송은 이 순서면 돼.
재윤: 제가 확인 안 해도 괜찮아요?
서린: 네가 오늘 확인해야 할 건 이게 아니잖아.
재윤: 맞는 답 말고, 제가 고른 이유요.
서린: 기억하네.
```

D-3 서린 정리 성공 시:

```text
재윤: 그래도 틀리면요?
서린: 틀리면 내가 고친다.
서린: 너는 네가 말해야 할 걸 말해.
```

서린은 점검표 일부를 건넨다.

```text
서린: 소무대 쪽은 폐장 후 철거 예정이야. 위험한 건 맞아.
재윤: 그럼 가지 말라고 하실 줄 알았는데요.
서린: 가도 된다고 말하는 것도 아니야.
재윤: 네.
서린: 알고 고르라는 거야.
```

D-3 서린 미정리 시:

```text
서린: 네가 정 못 고르겠으면, 내가 정해줄 수 있어.
재윤: 그게 편하긴 해요.
서린: 편한 게 정답은 아니야.
```

보정 선택지:

```text
[제가 고를게요]
[선배가 정해주세요]
```

성공 시:

```text
재윤: 제가 고를게요.
서린: 위험해도?
재윤: 위험하다는 걸 알고도, 제가 왜 가려는지 말해야 할 것 같아요.
서린: 그럼 가서 말해. 대신 변명하지 말고.
```

실패 시:

```text
재윤: 선배가 정해주세요.
서린: ...그럼 가지 마.
```

방송실 스피커에 폐장 멘트가 미리 울린다.

```text
방송: 지정된 동선을 따라 이동하십시오.
```

### 결과

성공:

- `SeoRinResolved = true` 또는 유지
- `AccidentClueSeoRin` 보강
- 폐장 후 선택 확신 상승

실패:

- `[선택하지 않는다]`, `[정해진 길을 따른다]` 계열 선택지 강화
- 서린 새드엔딩 진입 위험 상승

## 6. 낮 선택: 미루의 도움 인원 마무리를 확인한다

### 성공/보정 목적

- `MiRuResolved` 최종 확인 또는 보정
- 출입 제한 표지/고정끈 단서 보강
- 의존 잔류 여부 확정

### 대본

축제 도움 인원 대기 장소. 미루는 체크리스트를 품에 숨기지 않고 들고 있다.

대기 장소는 막힌 방이 아니라 복도 옆 임시 책상에 가깝다. 체크리스트, 명찰, 여분 안내 표지, 테이프와 끈, 분실물 박스가 보이고, 미루가 원하면 직접 나갈 수 있는 문이나 복도 동선이 보여야 한다.

D-3 미루 정리 성공 시:

```text
미루: 선배.
재윤: 어, 미루야.
미루: 저, 표지 확인하고 보고까지 했어요.
재윤: 잘했네.
미루: 그래도 마지막에 선배가 한 번 봐주시면—
재윤: 네가 먼저 본 게 맞아.
재윤: 내가 다시 정해주지 않아도 돼.
미루: 그럼 저, 혼자 한 거 맞죠?
재윤: 응. 네가 한 거야.
```

미루는 웃다가 울 것 같은 얼굴이 된다.

```text
미루: 그럼 저 이제 보고하러 갈게요.
재윤: 응.
미루: 선배도... 가야 하는 데 가세요.
```

D-3 미루 미정리 시:

```text
미루: 선배, 이것만 봐주시면 안 돼요?
재윤: 미루야.
미루: 마지막이잖아요. 방학식 지나면 멘토링 끝나니까.
```

보정 선택지:

```text
[네가 먼저 확인한 걸 믿는다]
[마지막이니까 대신 봐준다]
```

성공 시:

```text
재윤: 네가 먼저 확인한 걸 믿을게.
미루: 틀렸으면요?
재윤: 그럼 고치면 돼. 네가 한 게 없어지는 건 아니야.
미루: ...알겠어요.
```

미루는 체크리스트를 넘긴다.

```text
미루: 소무대 뒤편 표지는 다시 붙였고요.
미루: 고정끈은 임시로 묶인 흔적이 있었어요.
미루: 보고는 학생회 운영본부로 하면 된대요.
```

실패 시:

```text
재윤: 줘. 내가 볼게.
미루: 역시 선배가 봐줘야 안심돼요.
```

체크리스트 항목이 늘어난다.

```text
[마지막 확인]
[마지막의 마지막 확인]
[선배가 떠나지 않는지 확인]
```

### 결과

성공:

- `MiRuResolved = true` 또는 유지
- `AccidentClueMiRu` 보강
- 의존 도피 선택지 약화

실패:

- 미루 새드엔딩 위험 상승
- 하연에게 가는 선택이 누군가를 버리는 것처럼 흔들림

## 7. 낮 선택: 혼자 야외 소무대 쪽을 본다

### 성공/보정 목적

- 사고 단서 직접 보강
- 단, 하연 신뢰와 공유가 부족하면 오히려 빈자리를 강조

### 대본

재윤은 혼자 학교 뒤편으로 향한다. 낮의 소무대는 조용하다.

낮의 소무대는 아직 사고 장소처럼 보이지 않는다. 낮은 단차, 검은 전선, 임시 조명 프레임, 기울어진 출입 제한 표지는 보이지만, 햇빛 아래에서는 평범한 행사 시설처럼 지나칠 수 있어야 한다.

```text
소무대는 낮에도 조용했다.

표지는 제대로 붙어 있었고,
끈은 풀리지 않았고,
프레임은 아직 제자리에 있었다.
```

재윤은 점검표와 미루의 보고를 떠올린다.

```text
재윤: 전부 제자리에 있는데...
```

바람이 불고, 임시 조명 프레임이 아주 작게 운다.

하연 신뢰가 충분한 경우:

```text
재윤은 하연이 창밖으로 이곳을 바라보던 얼굴을 떠올렸다.

혼자 있고 싶은 거랑,
아무도 오지 않았으면 하는 건 달라.
```

하연 신뢰가 부족한 경우:

```text
전부 확인했는데도,
재윤은 여전히 누구에게 가려 했는지 알 수 없었다.
```

### 결과

성공:

- 사고 단서 보강
- `AccidentClueSeoRin`, `AccidentClueMiRu`, `AccidentClueHaYeon` 중 부족분 일부 보정

실패:

- 정보는 있지만 마음 연결이 부족
- 폐장 후 혼자 소무대에 설 위험 상승

## 8. 폐장 후 진입

축제 마지막 안내 방송이 끝난다.

```text
방송: 이상으로 축제 2일차 운영을 종료합니다.
방송: 각 반은 정리 후 지정된 동선으로 이동해 주시기 바랍니다.
```

운동장 쪽 불빛이 하나씩 꺼진다. 교실 장식이 내려가고, 카페 문구 카드도 떼어진다.

폐장 후 배경은 밝은 축제 중심부와 먼저 어두워지는 학교 뒤편을 대비한다. 운동장에는 아직 정리하는 학생과 접히는 천막이 남아 있지만, 소무대 쪽은 소리가 먼저 빠져나간다.

재윤은 문구 카드를 손에 든다.

```text
잠깐 앉아도 괜찮습니다.
```

재윤은 문구를 접어 주머니에 넣는다.

하연이 보이지 않는다.

처음에는 화장실, 교실 밖, 운동장 가장자리라고 생각한다. 하지만 어디에도 없다.

### 조건 정산

| 상태 | 폐장 후 인식 |
| --- | --- |
| 하연 신뢰 충분 + D-2 성공 | 하연이 조용한 소무대 쪽으로 갔음을 알아차림 |
| 하연 신뢰 부족 | 하연이 사라진 것을 늦게 알아차리거나, 알아차려도 동선을 모름 |
| 관계 정리 부족 | 하연에게 가는 마음이 죄책감/도피/정답 문제로 흔들림 |
| 사고 단서 부족 | 소무대가 위험 장소라는 것만 남고, 마음의 장소로 연결되지 않음 |

### 성공 쪽 내레이션

```text
하연이 보이지 않았다.

재윤은 낮에 하연이 바라보던 방향을 떠올렸다.
시끄러운 곳에서 한 걸음 물러난 자리.
들어와도 되고, 안 들어와도 되는 곳.
혼자 있고 싶은 것과,
아무도 오지 않았으면 하는 것은 다르다는 말.

학교 뒤편 야외 소무대.

이번에는,
누가 불러서 가는 게 아니었다.
```

성공 선택지:

```text
[하연에게 말하러 간다]
```

완전 조건 충족 시:

```text
[다친 기억까지 안고 하연에게 돌아간다]
```

### 실패 쪽 내레이션

```text
하연이 보이지 않았다.

재윤은 소무대 쪽이 신경 쓰였다.
왜인지는 알 수 없었다.

표지.
고정끈.
임시 프레임.
폐장 안내 방송.

장소와 물건은 떠올랐다.
하지만 그곳에 있어야 할 사람의 이름이 떠오르지 않았다.
```

실패 선택지:

```text
[소무대 쪽으로 간다]
[아무 일도 없었던 것으로 한다]
[선택하지 않는다]
[여기에 남는다]
```

## 9. 폐장 후 성공 분기: 하연에게 말하러 간다

### 장소

학교 뒤편 야외 소무대.

낮에는 축제의 일부였지만, 밤에는 조용하다. 무대 위 조명은 꺼져 있고, 임시 프레임은 아직 철거되지 않았다. 바람이 불면 금속이 작게 운다.

하연은 무대 위가 아니라, 무대 옆 낮은 계단 근처에 서 있다. 위험한 구조물 안쪽으로 깊게 들어가 있지는 않다. 그녀는 재윤을 부른 적이 없다.

배경 핵심 소품은 낮은 단차, 검은 전선, 임시 조명 프레임, 출입 제한 표지, 철거 대기 장비 박스다. 진엔딩 구도에서는 이 소품들이 두 사람을 위협하는 장치로만 보이지 않고, 재윤이 되찾은 사고 기억의 조각으로도 읽혀야 한다.

### 도착

```text
하연: 어떻게 알았어?
재윤: 네가 있을 것 같았어.
```

하연은 소무대 쪽을 바라보다가 재윤 쪽으로 시선을 돌린다.

```text
하연: 내가 부른 것도 아닌데.
재윤: 그래서 온 거야.
재윤: 이번엔 내가 먼저 와야 할 것 같아서.
```

하연은 바로 대답하지 않는다.

```text
하연: 지금은 알아?
재윤: 응.
하연: 뭘?
재윤: 누구한테 말하러 왔는지.
```

### 재윤 선고백

```text
재윤: 오래된 쪽으로 돌아가면 편했어.
재윤: 맞는 답을 찾는 척하면 덜 무서웠고.
재윤: 누가 나를 필요로 하면,
재윤: 내가 뭘 원하는지 안 봐도 됐어.
```

D-3 정리 결과에 따라 짧은 회상 컷이 끼어든다.

- 소하: 오래된 골목에서 돌아서며 `지금 안 가면 또 늦잖아.`
- 서린: 방송실에서 `네가 고른 답이면 끝까지 봐.`
- 미루: 체크리스트를 들고 `저도 제 거 해볼게요.`

```text
재윤: 그런데 그러면,
재윤: 나는 계속 아무 데도 못 가.

재윤: 이번에는 너한테 가고 싶었어.
재윤: 누가 시켜서도 아니고,
재윤: 네가 먼저 말해줘서도 아니고.

재윤: 내가 고른 거라고 말하고 싶어서.
```

재윤은 한 번 숨을 고른다.

```text
재윤: 하연아.
재윤: 나는 너를 좋아해.
```

### 선택 정리 변주

조건에 따라 고백 전후에 한 줄씩 추가할 수 있다.

| 정리한 축 | 추가 대사 |
| --- | --- |
| 소하 정리 | `예전으로 돌아가 숨고 싶었던 마음을 지나서 왔어.` |
| 서린 정리 | `맞는 답이 아니라, 내가 고른 답이라고 말하려고 왔어.` |
| 미루 정리 | `누군가에게 필요한 사람인 척하면서 내 마음을 미루지 않으려고 왔어.` |
| D-2 성공 | `네가 조용한 데서 숨을 고르는 사람이라는 걸, 이제는 조금 알아.` |

### 하연 마음 확인

하연은 시선을 피하지 않는다. 다만 바로 웃지도 않는다.

```text
하연: 나도 말하려고 했어.
재윤: 오늘?
하연: 응. 끝나고, 조금 조용해지면.
```

재윤의 표정이 흔들린다.

```text
재윤: 왜 말 안 했어?
하연: 네가 올 수 있으면... 좋겠다고 생각했어.
재윤: 내가?
하연: 응.
```

하연은 아주 천천히 말한다.

```text
하연: 네가 나를 기다려준 적이 있었잖아.
하연: 억지로 끌고 오지 않고,
하연: 들어와도 된다고 말해준 적.

하연: 그래서 나도 기다렸어.
하연: 네가 돌아보는 걸.
하연: 네가 네 발로 오는 걸.
```

```text
재윤: 많이 늦었지.
하연: 늦었어.
재윤: 미안.
하연: 그 말보다...
재윤: 응.
하연: 와줘서 다행이야.
```

하연도 말한다.

```text
하연: 나도... 좋아해.
하연: 말하는 게 늦었지만,
하연: 없던 말로 만들고 싶지는 않았어.
```

### 사고 기억 회복

마음 확인 직후, 세계가 잠깐 조용해진다.

폐장 안내 방송의 끝부분이 다시 들린다.

```text
방송: ...정리 후 지정된 동선으로 이동해 주시기 바랍니다.
```

재윤의 시야에 단서들이 겹친다.

```text
조명 프레임.
출입 제한 표지.
풀리지 않은 고정끈.
낮게 남은 소무대의 단차.

전부 같은 자리에 있었다.
```

하연이 한 걸음 움직인다. 바람이 불고, 임시 조명 프레임이 크게 흔들린다.

```text
하연: 재윤아?
```

재윤은 이미 알고 있다. 이 다음에 무슨 일이 오는지.

```text
재윤: 하연아, 뒤로.
```

그는 하연을 밀어내거나 감싸려 손을 뻗는다. 사고는 사라지지 않는다.

내레이션:

> 사고는 사라지지 않았다.  
> 아픈 기억도 사라지지 않았다.  
> 다만 이번에는, 재윤이 어디로 가려 했는지 알았다.  
> 하연에게.  
> 말하기 위해서.

선택지:

```text
[다친 기억까지 안고 하연에게 돌아간다]
```

선택 후 암전.

### 성공 결과

- 사고 기억 수용
- 일요일 오후 병실로 점프
- D+3 월요일 방학식 도달
- 진엔딩 크레딧 조건 충족

## 10. 폐장 후 실패 분기: 혼자 소무대에 선다

### 실패 조건

아래 중 하나 이상이 결정적으로 부족할 때 발생한다.

- `HaYeonTrust` 부족
- `HaYeonPaceRead` 부족
- D-2 예감 실패
- 기존 관계 정리 부족
- 사고 단서와 하연 기억을 연결하지 못함

### 장소

같은 학교 뒤편 야외 소무대.

하지만 하연은 없다. 하연이 서 있던 계단 옆 자리도 비어 있다. 바람 소리와 폐장 방송만 남는다.

실패 구도에서는 같은 소품이 남아도 의미가 빠져 있어야 한다. 전선, 단차, 표지, 조명 프레임은 선명하지만, 하연이 머물던 자리와 말하려던 이유가 비어 있어 재윤이 장소만 따라온 느낌을 준다.

### 대본

```text
축제 마지막 안내 방송이 끝났다.
운동장 쪽 불빛이 하나씩 꺼졌다.

재윤은 학교 뒤편 야외 소무대 앞에 서 있었다.
```

재윤은 주변을 본다.

```text
재윤: ...왜 여기까지 왔지?
```

표지, 고정끈, 프레임, 단차가 보인다.

```text
표지는 제대로 붙어 있었다.
끈은 묶여 있었다.
프레임은 제자리에 있었다.

그러니까 이상했다.

전부 제자리에 있는데,
재윤만 왜 여기 서 있는지 알 수 없었다.
```

기억 조각이 떠오르려다 사라진다.

```text
누군가의 목소리.
말하려던 문장.
창밖을 보던 옆얼굴.

하지만 이름이 떠오르지 않았다.
```

실패 선택지:

```text
[아무 일도 없었던 것으로 한다]
[선택하지 않는다]
[여기에 남는다]
```

선택지와 상관없이 하연이 빠진 사고 기억으로 수렴한다. 선택지는 재윤의 회피 양상을 보여주기 위한 연출이다.

```text
재윤: 누가...
재윤: 누가 있었던 것 같은데.
```

폐장 안내 방송의 끝부분이 멀리서 다시 울리는 것 같다.

```text
방송: ...이동해 주시기 바랍니다.
```

금속이 삐걱인다.

```text
재윤: 나는...
```

발밑이 비는 감각.

암전.

### 실패 후 루프

사고 직후를 길게 설명하지 않는다.

```text
눈을 떴을 때,
창밖은 월요일 아침이었다.

교실에는 아직 오지 않은 이름이 있었고,
재윤은 다시 같은 하루 앞에 서 있었다.
```

내부적으로 D-25 월요일로 복귀한다. 게임 내 표기는 `월요일 아침`으로 처리한다.

### 실패 결과

- 하연 실패 루프
- D+3 방학식 미도달
- 완성된 엔딩 크레딧 없음
- 다음 루프에서 하연 관련 기억 일부가 더 흐려지거나, 소무대 단서만 강하게 남음

## 11. D-Day 분기 판정표

상세 수치 기준: [D-1/D-Day 조건 수치화](./12_D-1_D-Day_조건_수치화.md)

요약 우선순위:

| 우선순위 | 조건 | 분기 |
| --- | --- | --- |
| 1 | 히든 붕괴 조건 충족 | 히든 세계 붕괴 엔딩 |
| 2 | D-2 예감 실패, `HaYeonTrust < 6`, `HaYeonPaceRead < 3` | 하연 실패 루프 |
| 3 | `SoHaResolved == false` + `SoHaPressure == 3` | 소하 새드엔딩 |
| 4 | `SeoRinResolved == false` + `SeoRinControl == 3` | 서린 새드엔딩 |
| 5 | `MiRuResolved == false` + `MiRuDependence == 3` | 미루 새드엔딩 |
| 6 | 기존 관계 미해결이 남음 | 기존 관계 미정리 루프 또는 약화 새드엔딩 |
| 7 | `AccidentClueScore < 3` 또는 `AccidentClueHaYeon == false` | 사고 의미 미연결 루프 |
| 8 | `Avoidance >= 4` | 회피 루프 |
| 9 | 모든 필수 조건 충족 | 진엔딩 |

동점 처리:

- 하연 조건 부족은 항상 다른 새드엔딩보다 우선한다.
- 기존 관계 미해결이 여럿이면 가장 압박 수치가 높은 인물의 새드엔딩 또는 루프로 보낸다.
- 사고 단서 부족은 기존 관계 정리가 끝났지만 기억 의미 연결이 부족할 때 발생한다.

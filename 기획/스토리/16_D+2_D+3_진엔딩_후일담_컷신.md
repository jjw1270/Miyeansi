# D+2/D+3 진엔딩 후일담 컷신

## 문서 목적

이 문서는 D-Day 진엔딩 성공 이후, 플레이어가 처음으로 금요일 다음의 현실 시간을 통과하는 장면을 실제 컷신 단위로 정리한 제작 정본이다.

- 본편 감정 원고 기준: [전체 스토리](./00_전체_스토리.md)
- 엔딩 의미 기준: [엔딩과 루프](./05_엔딩과_루프.md)
- 날짜 배치 기준: [날짜별 진행 달력](./06_날짜별_진행_달력.md)
- D-Day 직전 연결 기준: [D-Day 스크립트 분할](./13_D-Day_스크립트_분할.md)
- 이후 크레딧 기준: [진엔딩 크레딧 콘티 초안](./11_진엔딩_크레딧_콘티_초안.md)
- 장소 감정 기준: [장소와 배경 묘사](./14_장소와_배경_묘사.md)
- 배경 에셋 기준: [배경 에셋 제작 리스트와 이미지 프롬프트](./15_배경_에셋_프롬프트.md)

이 문서는 소설형 원고를 다시 길게 반복하지 않는다. 실제 구현과 연출에서 바로 쓸 수 있도록 장면 ID, 진입 조건, 배경, 음악, 핵심 대사, 플래그를 기준으로 정리한다.

## 전환 원칙

| 구간 | 표시 방식 | 의미 |
| --- | --- | --- |
| D-Day 금요일 밤 | 사고 직후 암전 | 재윤이 하연과 자신의 마음을 기억한 채 현실로 넘어가는 문턱 |
| D+1 토요일 | 날짜 카드 없음 | 현실에서는 혼수 상태가 이어지지만, 플레이어에게 별도 플레이 날짜로 주지 않음 |
| D+2 일요일 | 병실 기상 컷신 | 재윤이 처음으로 사고 이후의 현실을 받아들이는 장면 |
| D+3 월요일 | 방학식 컷신 | 실패 루프가 도달하지 못한 월요일을 진엔딩에서만 증명하는 장면 |
| 크레딧 | 음악과 짧은 애니메이션 컷 | 방학 동안 두 사람이 계속 만날 것을 암시하는 보상 연출 |

비진엔딩에서는 이 문서의 D+2/D+3 컷신을 사용하지 않는다. 소하/서린/미루 새드엔딩과 하연 실패 루프는 각각 가짜 크레딧, 짧은 암전, 루프 복귀 연출로 처리한다.

## TE_00_TimeSkip

```text
@enter IsDDayTrueEnding == true
@date_card none
@bg black
@music none
@sfx heart_monitor_low, distant_corridor
```

토요일은 날짜 카드로 보여주지 않는다. 검은 화면 위로 병원 기계음, 멀리서 닫히는 문소리, 하연이 숨을 참았다가 내쉬는 소리만 짧게 들린다.

핵심 연출:

- `D+1 토요일` 텍스트를 띄우지 않는다.
- 플레이어가 루프가 다시 시작된 줄 착각할 만큼 길게 끌지 않는다.
- 검은 화면은 공포가 아니라 현실로 돌아오는 완충이다.
- 마지막에 하연의 손이 재윤의 손가락을 살짝 붙잡는 짧은 컷으로 병실을 예고한다.

나레이션 후보:

```text
하루가 지나갔다.
하지만 이번에는, 아무도 그 하루를 다시 처음으로 돌려놓지 않았다.
```

정산:

```text
TE_Sat = true
```

## TE_01_Wake

```text
@enter TE_Sat == true
@date_card "D+2 일요일 오후"
@bg BG_HOSPITAL_ROOM_DAY
@music true_ending_wake_soft
@sfx heart_monitor_soft, curtain_wind
```

재윤은 병실의 흰 천장과 커튼 사이로 들어오는 햇빛을 먼저 본다. 루프의 교실 천장이 아니라는 사실이 가장 먼저 와 닿는다. 손등에는 주삿바늘 자국이 있고, 몸은 무겁지만 크게 다친 통증은 없다.

하연은 침대 옆 의자에 앉아 있다. 교복 셔츠 위에 가벼운 가디건을 걸치고, 잠을 거의 못 잔 얼굴로 재윤의 손 근처를 바라보고 있다. 하연은 재윤이 깨어난 것을 보고도 바로 울거나 달려들지 않는다. 먼저 숨을 고르고, 재윤이 놀라지 않게 아주 작은 목소리로 말한다.

핵심 대사:

```text
하연: 깼어?
재윤: ...여기.
하연: 병원이야.
재윤: 오늘... 금요일 아니지.
하연: 일요일이야. 오후.
재윤: 일요일.
하연: 응. 너, 토요일 내내 안 깼어.
```

재윤은 잠깐 눈을 감는다. D-25 월요일 교실이 아니라, 일요일 병실이라는 사실을 받아들이는 데 시간이 걸린다.

```text
재윤: 다시 안 돌아갔네.
하연: ...응.
재윤: 나, 기억해.
하연: 뭘?
재윤: 네가 거기 있었던 거. 내가 너한테 가려고 했던 거. 그리고... 내가 말했던 거.
```

정산:

```text
HasSeenTrueEndingWake = true
```

## TE_02_Real

```text
@enter HasSeenTrueEndingWake == true
@bg BG_HOSPITAL_ROOM_DAY
@music true_ending_reality_confirm
@sfx nurse_cart_far, curtain_wind
```

이 장면은 설명용 대사가 길어지지 않게 한다. 의료 정보는 최소화하고, 재윤과 하연이 서로의 기억을 확인하는 데 집중한다.

핵심 정보:

- 재윤은 큰 외상은 없지만 사고 충격 뒤 의식이 돌아오지 않았다.
- 하연은 크게 다치지 않았고, 병원과 학교를 오가며 기다렸다.
- 경찰/학교 조사는 이미 어른들이 처리 중이며, 이 컷신의 중심이 아니다.
- 중요한 것은 `루프가 끝났는지`가 아니라 `둘 다 그 밤을 기억하는지`다.

핵심 대사:

```text
재윤: 너는?
하연: 난 괜찮아. 정말로.
재윤: 거짓말 같아.
하연: 거짓말이면 좋겠다고 생각한 적은 있어.
재윤: ...
하연: 근데 네가 깨어났으니까, 이제 그 생각은 안 하려고.
```

하연은 자신의 마음을 재윤 대신 정리해주지 않는다. 대신 재윤이 기억을 끝까지 말할 수 있게 기다린다.

```text
재윤: 무서웠어.
하연: 응.
재윤: 다친 게 무서웠던 게 아니라... 내가 그걸 없던 일로 만들까 봐.
하연: 이제 안 그럴 거야?
재윤: 응. 이번엔 기억하고 왔어.
```

정산:

```text
TE_Real = true
HasAcceptedAccidentMemory = true
```

## TE_03_Confess

```text
@enter TE_Real == true
@bg BG_HOSPITAL_ROOM_DAY
@music true_ending_confession_small
@sfx heart_monitor_soft
```

이 장면은 D-Day 폐장 후 고백의 반복이 아니라, 현실에서 같은 마음을 다시 확인하는 장면이다. 감정은 크지만 연출은 작게 둔다. 하연이 울음을 터뜨리는 장면보다, 손을 놓지 않는 장면이 더 중요하다.

핵심 대사:

```text
재윤: 하연아.
하연: 응.
재윤: 좋아해.
하연: ...
재윤: 내일도, 다음 주도, 방학에도. 도망가지 않고 말하고 싶어.
하연: 병원 침대에서 그런 말 하는 거, 되게 이상한 거 알아?
재윤: 알아.
하연: 그런데 나도... 좋아해.
```

하연은 `나도`를 빨리 말하지 않는다. 잠깐 웃고, 눈가가 젖은 뒤에야 말한다. 하연의 고백은 극적인 선언보다 `이제는 미루지 않는 대답`에 가깝다.

```text
하연: 나도 안 미룰게.
재윤: 응.
하연: 그러니까 너도, 혼자 정리 끝낸 척하지 마.
재윤: 안 할게.
```

정산:

```text
TE_Confess = true
HayeonReal = true
```

## TE_04_Depart

```text
@enter TE_Confess == true
@date_card "D+3 월요일 아침"
@bg BG_HOSPITAL_CORRIDOR
@music true_ending_monday_light
@sfx elevator_chime, corridor_steps
```

월요일 아침, 재윤은 퇴원까지는 아니더라도 잠시 외출 허가를 받는다. 과장된 기적처럼 보이지 않게, 보호자 동행과 짧은 참석만 허용된 상태로 둔다.

핵심 대사:

```text
하연: 무리하면 바로 돌아오는 거야.
재윤: 알았어.
하연: 진짜로.
재윤: 응. 진짜로.
하연: 방학식이 그렇게 중요해?
재윤: 월요일이 왔으니까.
```

재윤이 방학식에 가려는 이유는 행사가 중요해서가 아니다. 실패 루프에서 닿지 못했던 다음 월요일이 실제로 왔다는 것을 자기 눈으로 확인하기 위해서다.

정산:

```text
HasReachedTrueEndingMonday = true
TE_Depart = true
```

## TE_05_Class

```text
@enter HasReachedTrueEndingMonday == true
@bg BG_CLASSROOM_CEREMONY
@music true_ending_school_return
@sfx classroom_murmur, summer_cicada_far
```

교실은 축제 뒤의 들뜸과 방학식 전의 느슨함이 섞여 있다. 장식 일부가 아직 떼어지지 않았고, 칠판 구석에는 `방학식` 안내가 남아 있다. 재윤이 문을 열고 들어오면 교실이 한순간 조용해진다.

핵심 연출:

- 카메라는 재윤의 얼굴보다 교실의 `월요일 오전` 공기를 먼저 보여준다.
- 축제 장식은 남아 있지만 밤의 사고 분위기는 사라져 있다.
- D-25 월요일 교실과 비슷한 구도에서 시작하되, 하연의 자리가 비어 있지 않다.

핵심 대사:

```text
반 친구: 유재윤?
다른 친구: 야, 괜찮아?
재윤: 오래는 못 있어.
반 친구: 누가 오래 있으래. 앉아.
```

정산:

```text
TE_Class = true
```

## TE_06_Friends

```text
@enter TE_Class == true
@bg BG_CLASSROOM_CEREMONY
@music true_ending_friends_soft
@sfx classroom_murmur_low
```

소하, 서린, 미루는 진엔딩에서 연애 경쟁자가 아니라 재윤이 더 이상 숨지 않는 관계의 증거로 등장한다. 각 인물의 반응은 짧고, 성격에 맞아야 한다.

소하:

```text
소하: 너 진짜 사람 놀라게 하는 데 재능 있다.
재윤: 미안.
소하: 미안하면 방학 때 연락 씹지 마.
재윤: 안 씹을게.
소하: 그 말, 녹음하고 싶네.
```

서린:

```text
서린: 출석 처리보다 먼저 양호실에 들러야 하는 상태로 보이는데.
재윤: 오래 안 있을 거야.
서린: 그럼 괜찮아. 네가 오늘 여기에 온 이유는, 오래 버티는 게 아니라 왔다는 사실이니까.
```

미루:

```text
미루: 선배, 진짜 괜찮아요?
재윤: 응. 걱정시켜서 미안.
미루: 그럼 방학 때... 멘토링 끝나도 가끔 인사해도 돼요?
재윤: 당연하지.
미루: 그럼 됐어요.
```

정산:

```text
TE_Friends = true
SohaAck = true
SeorinAck = true
MiruAck = true
```

## TE_07_Hayeon

```text
@enter TE_Friends == true
@bg BG_CLASSROOM_CEREMONY
@music true_ending_hayeon_soft
@sfx classroom_murmur_low, summer_cicada_far
```

하연은 교실 뒤쪽 창가 근처에 있다. 재윤이 먼저 찾아가도 되고, 하연이 조용히 다가와도 된다. 중요한 것은 둘 중 누구도 더 이상 `우연처럼 지나가는 척`하지 않는다는 점이다.

핵심 대사:

```text
하연: 진짜 왔네.
재윤: 응.
하연: 어제보다 얼굴 안 좋아 보이는데.
재윤: 월요일 얼굴이라서 그래.
하연: 그런 농담은 방학 때도 계속할 거야?
재윤: 네가 들어주면.
하연: 그럼 조금만.
```

둘은 큰 약속을 잡지 않는다. 대신 방학 동안 계속 만날 가능성을 작은 일정들로 나눈다.

```text
재윤: 병원 다시 가는 날, 같이 와줄래?
하연: 응.
재윤: 도서관도.
하연: 응.
재윤: 카페도.
하연: 천천히.
재윤: 응. 천천히.
```

정산:

```text
TE_Hayeon = true
TE_SummerSeed = true
```

## TE_08_Ceremony

```text
@enter TE_Hayeon == true
@bg BG_CLASSROOM_CEREMONY
@music true_ending_ceremony_small
@sfx school_speaker, chair_noise
```

방학식은 강당 대규모 연출보다 교실 방송식으로 처리하는 편이 좋다. 재윤의 몸 상태상 오래 서 있는 장면을 피하고, `학교가 평범하게 다음 계절로 넘어간다`는 감각을 살린다.

핵심 연출:

- 방송 스피커에서 방학식 멘트가 들린다.
- 학생들이 의자를 밀고, 책상을 정리한다.
- 축제 포스터가 떼어지고, 방학 안내 프린트가 나눠진다.
- 재윤은 하연의 손을 잡지 않아도 된다. 대신 같은 프린트를 같이 들여다본다.

정산:

```text
HasSeenTrueEndingCeremony = true
HasSummerStarted = true
```

## TE_09_Promise

```text
@enter HasSeenTrueEndingCeremony == true
@bg BG_CORRIDOR_NORMAL
@music true_ending_after_school
@sfx corridor_steps, summer_wind
```

방학식 직후, 복도와 학교 앞은 방학의 시작으로 느슨하다. 재윤은 오래 걷지 못해 중간에 멈춘다. 하연은 재촉하지 않고 옆에서 속도를 맞춘다.

핵심 대사:

```text
하연: 병원부터 가야지.
재윤: 응.
하연: 그리고?
재윤: 네가 괜찮으면, 도서관.
하연: 응.
재윤: 카페.
하연: 응.
재윤: 산책은... 조금 나중에.
하연: 천천히 와.
재윤: 기다릴 거야?
하연: 안 가.
```

이 장면의 핵심은 `연애가 완성되었다`가 아니라 `다음 약속을 미루지 않게 되었다`다. 방학의 긴 시간을 한 번에 소비하지 않고, 병원·도서관·카페·산책 같은 작은 장면으로 크레딧에 넘긴다.

정산:

```text
TE_AfterSchool = true
TE_Promise = true
```

## TE_10_Credit

```text
@enter TE_Promise == true
@bg BG_SUMMER_WALKWAY
@music true_ending_credit_intro
@sfx summer_cicada, soft_steps
```

하연과 재윤이 학교를 나서는 뒷모습을 길게 보여주지 않는다. 너무 완성된 데이트 장면으로 보이면 여운이 닫힌다. 대신 햇빛, 느린 걸음, 같은 방향으로 접히는 그림자 정도만 남긴다.

연결 문장 후보:

```text
금요일 다음에는, 월요일이 있었다.
그리고 월요일 다음에는, 우리가 아직 말하지 않은 여름이 있었다.
```

정산:

```text
CanEnterTrueEndingCredit = true
TE_Done = true
```

## 배경/연출 에셋 기준

| 에셋 ID | 사용 장면 | 제작 기준 |
| --- | --- | --- |
| `BG_HOSPITAL_ROOM_DAY` | `TE_01_Wake`~`TE_03_Confess` | 깨끗하지만 차갑지 않은 일요일 오후 병실. 흰 커튼, 낮은 햇빛, 침대 옆 의자, 심박 모니터 |
| `BG_HOSPITAL_CORRIDOR` | `TE_04_Depart` | 월요일 아침 병원 복도. 과한 긴장보다 조심스럽게 현실로 나가는 느낌 |
| `BG_CLASSROOM_CEREMONY` | `TE_05_Class`~`TE_08_Ceremony` | 축제 장식 일부와 방학식 안내가 같이 남은 2학년 5반 교실 |
| `BG_CORRIDOR_NORMAL` | `TE_09_Promise` | 방학식 직후의 밝은 복도. 사람은 줄고 여름빛이 강해짐 |
| `BG_SUMMER_WALKWAY` | `TE_10_Credit` | 학교 밖으로 이어지는 여름 보도. 크레딧 진입용 짧은 애니메이션 컷 후보 |

`BG_HOSPITAL_ROOM_DAY`, `BG_HOSPITAL_CORRIDOR`, `BG_CLASSROOM_CEREMONY`, `BG_SUMMER_WALKWAY`는 15번 배경 에셋 문서에서 실제 프롬프트와 제작 체크리스트를 관리한다. 이 문서에서는 사용 맥락과 감정 기능만 확정한다.

# 기획 문서 안내

이 폴더는 프로젝트의 스토리 기획과 시스템 기획을 함께 관리한다.

## 문서 운용 기준

- 원본 정본은 Markdown 문서다.
- `통합_기획서.html`은 원본 Markdown을 모아 만든 읽기/검색용 생성물이다.
- 문서 역할, 정본/요약/생성물 구분, 오래된 문서 삭제 기준은 [기획 문서 정본 가이드](./문서_정본_가이드.md)를 따른다.

## 통합 웹 기획서

- [통합 기획서 웹 뷰](./통합_기획서.html)
  - `기획/` 아래 Markdown 기획서를 하나의 단일 HTML로 묶은 문서
  - 루트 `../index.html`과 동일한 내용의 보조 HTML
  - 좌측 문서 목록, 카테고리 필터, 전체 검색, 선택 문서 목차 지원
  - 모든 문서를 아래로 길게 펼치지 않고, 선택한 문서 하나만 중앙 뷰어에 표시
  - 스토리와 시스템 기획을 함께 확인할 때 사용

주의:

- `통합_기획서.html`은 직접 수정하지 않는다.
- Markdown 원본을 수정한 뒤 빌드 스크립트로 재생성한다.

## 웹 기획서 재생성

Markdown 원본을 수정한 뒤 아래 명령으로 통합 웹 문서를 다시 생성한다.

```powershell
python 기획/build_planning_web.py
```

기준 문서는 각 Markdown 원본이며, `통합_기획서.html`은 탐색과 검토를 위한 웹 뷰다.
